# Salona Products – Frontend Task

A paginated product listing built with **React**, **TanStack Query (React Query)**, and **Tailwind CSS**.  
Focus areas: data fetching with caching, resilient error states, skeleton loading, accessible pagination, and a clean, responsive UI.

---

## 🚀 Quick Start

### Requirements
- **Node.js 20 LTS** (recommended)
- One package manager: **npm / pnpm / yarn**

### Install & Run
```bash
# 1) Install dependencies
npm ci

# 2) Start the dev server
npm run dev
